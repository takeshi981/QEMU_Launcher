## QEMU_Launcher
A launcher for QEMU VM (PyQt6)

Make sure qemu is installed and configured in environment variable (path)

# Architecture-aware launcher  
*Automatically detects and organizes available QEMU binaries by architecture.*  
# VM Configuration wizard  
*Step-by-step assistant to define RAM, CPU, disk image, BIOS, and network settings.*  
# Snapshot mode toggle  
*Launch VMs in temporary mode for safe testing.*  
# System theme integration  
*Adapts to light/dark mode based on your OS settings.*  
# Persistent configuration  
*Save and load VM profiles in JSON format.*  
# Supports English, Italian and French language
